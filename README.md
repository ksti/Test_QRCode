
# 注意，仅使用于iOS8及以上版本，因为用到了iOS8的API做从相册解析二维码，iOS7貌似只能用摄像头采集解析二维码
# ios 二维码、条形码 

## 简述
**iOS扫码封装 objective-c版本 封装ios系统API 用到iOS8API
- 扫码界面效果封装
- 二维码、条形码
- 相册获取图片后识别
- 生成二维码及条形码
- 自动检测环境光开启闪关灯

- 摇一摇封装在了 MyQRCodeViewController 里
- 支持摇动时长的检测

**其他设置参数自定义效果**

- 扫码框样式1：小屏模式
- 扫码框样式2：全屏模式
- 可在二维码上加logo
- 可配置二维码生成颜色
